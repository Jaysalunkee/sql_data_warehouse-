# 🏢 SQL Data Warehouse

An end-to-end SQL Data Warehouse project demonstrating ETL pipelines, dimensional modeling, data transformation, and analytical reporting using SQL Server.

---

## 📖 Overview

This project demonstrates how to design and build a modern SQL Data Warehouse from raw data to business-ready analytics. It follows industry best practices for data warehousing, including ETL processes, dimensional modeling, and analytical querying.

---

## 🎯 Objectives

- Build a centralized Data Warehouse
- Clean and transform raw data
- Design a Star Schema
- Create Fact and Dimension tables
- Perform business analysis using SQL
- Generate meaningful insights for decision-making

---

## 🛠️ Technologies Used

- SQL Server
- T-SQL
- SQL Server Management Studio (SSMS)

## 🔄 ETL Workflow

1. Extract raw data
2. Load data into staging tables
3. Clean and validate data
4. Transform data
5. Load Dimension tables
6. Load Fact tables
7. Run analytical queries
